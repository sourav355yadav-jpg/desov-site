1. TECH STACK & ARCHITECTURE
____________________________________________________________
Framework & Routing
Next.js 14 (App Router or Pages Router — uses _next/image, standard SSG/SSR)
React 18 with functional components and hooks
File-based routing: /, /studio, /work, /work/[slug], /contact, /careers/[role]
TypeScript (highly likely given the Awwwards-level build quality)
CMS
Sanity.io — headless CMS for all project/work content
Project images served from: cdn.sanity.io/images/9xh2olrq/production/
Dataset ID: 9xh2olrq (visible in all Sanity image URLs)
Content types: projects (slug, title, services, description, gallery, video, websiteLink, nextProject)
Animation Libraries
GSAP (GreenSock Animation Platform) — primary animation engine
GSAP ScrollTrigger — scroll-driven animations
Framer Motion or custom GSAP — page transition curtain
Lenis or Locomotive Scroll — smooth inertial scrolling
Possible: Three.js for 3D render elements in hero
Styling
Tailwind CSS or custom CSS Modules — utility-first approach
Theme color: #27b7a5 (teal) — set in meta theme-color
Dark background throughout — near-black (#0d0d0d to #111111)
White/off-white text (#EEEEEE or #FFFFFF)
Media
Next.js <Image> component for all images — quality=75, width=3840
Custom HTML5 <video> with controls — used in Studio reel and project video sections
Video thumbnails with overlay play button
Deployment
Vercel (implied by Next.js and performance level)
Domain: sohub.digital
OG images: /og-images/home.png, studio.png, work.png, contact.png

2. GLOBAL DESIGN SYSTEM
____________________________________________________________
Color Palette
▶ ANIMATION: Color Tokens

Typography
HEADINGS
Font: likely 'Neue Haas Grotesk', 'Helvetica Neue', or 'Inter' — ultra-clean grotesque
Hero H1: ~96–120px, font-weight 400 or 500, line-height 0.95, letter-spacing -0.03em
Section H2: ~48–64px, font-weight 400, tight tracking
Sub-heading H3: ~28–36px
All uppercase navigation items
BODY
Body text: ~16–18px, font-weight 300–400, color #888888 or #EEEEEE depending on context
Line height: 1.5–1.7 for body copy
SPECIAL
Animated character-split headings — each letter/word wrapped in a span and revealed individually
'Your story builds our history.' — 2-line hero text, word-by-word slide-up reveal
Spacing System
Generous whitespace — sections have 120–200px vertical padding
Max content width: ~1400px, centered
Side padding: 40–80px on desktop, 24px on mobile
Grid: mostly CSS Grid or Flexbox, no fixed column framework
Cursor
Custom cursor: small ~8px solid teal dot replaces default OS cursor
Follower ring: larger ~40px circle follows with 0.08–0.15 lerp (lag)
On hoverable elements: dot disappears, ring expands to ~80px, possibly shows label text
On links/project cards: cursor transforms to show text like 'View' inside the ring
On video elements: cursor shows 'Play' text inside ring
Implemented via mousemove event + requestAnimationFrame + CSS transform translate3d
